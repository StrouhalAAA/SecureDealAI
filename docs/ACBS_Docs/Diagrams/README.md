## Použití

### VS Code / IDE Preview
- Otevři `.md` soubor a použij Markdown Preview (Cmd+Shift+V)
- Vyžaduje rozšíření podporující Mermaid (např. "Markdown Preview Mermaid Support")

### mermaid.live
1. Otevři https://mermaid.live/
2. Zkopíruj obsah mezi \`\`\`mermaid a \`\`\`
3. Vlož do editoru

## Poznámky

- Každý soubor obsahuje nadpis, popis a Mermaid diagram
- Styling classes jsou součástí každého diagramu
- Diagramy používají emoji pro lepší orientaci
